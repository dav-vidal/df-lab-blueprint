# Unit Generator

INPUT:
  CREATURE: TOKEN (e.g., DWARF)
  CASTE: optional (MALE|FEMALE) if absent pick uniformly among defined castes.
  RNG_SEED: optional
PROCESS:
  - Load Configs/CreatureIndex.txt → creature file.
  - For each attribute with 7 cutoffs C=[c1..c7]:
      roll percentile → select bucket →
      sample uniformly within bucket range.
  - Body mods: sample HEIGHT_% and BROADNESS_% uniformly; set BASE_SIZE_CM3.
  - Initialize SKILLS: each SKILL_TOKENS → 0.
OUTPUT: UNIT_RAW (no IDs required here)
FALLBACK:
  - If creature file missing/invalid → NORMAL_FALLBACK (clamped).
